Custom Themes

Create a theme in a separate CSS file

Define your theme with [data-ui-theme="name"] selectors. Only override the properties you need — everything else inherits from the defaults. Hover, active, and light variants are derived automatically via color-mix().

Show code 
/* my-theme.css */

/* Light mode overrides */
[data-ui-theme="my-theme"] {
  --ui-primary: #6d28d9;
  --ui-primary-text: #ffffff;
  --ui-success: #059669;
  --ui-danger: #dc2626;

  --ui-bg: #faf5ff;
  --ui-bg-subtle: #f3e8ff;
  --ui-surface: #ffffff;
  --ui-border: #e0c6f5;
  --ui-text: #2e1065;
  --ui-text-muted: #7c3aed;
}

/* Dark mode overrides */
[data-ui-theme="my-theme"][data-theme="dark"] {
  --ui-primary: #a78bfa;
  --ui-primary-text: #ffffff;
  --ui-success: #34d399;
  --ui-danger: #fb7185;

  --ui-bg: #1a0533;
  --ui-bg-subtle: #130425;
  --ui-surface: #2a1045;
  --ui-border: #3d1a60;
  --ui-text: #e9d5ff;
  --ui-text-muted: #b08adb;
}

Include & activate

Show code 
<!-- 1. Include after ui.css -->
<link rel="stylesheet" href="ui.css">
<link rel="stylesheet" href="my-theme.css">

<!-- 2. Activate via attribute -->
<html data-ui-theme="my-theme">

<!-- Or toggle via JavaScript -->
<script>
// Apply theme
document.documentElement.setAttribute('data-ui-theme', 'my-theme');

// Revert to default
document.documentElement.removeAttribute('data-ui-theme');
</script>

Available properties

These are the properties you can override in a theme file. All are optional.

Show code 
/* Semantic colors (hover/active/light auto-derived) */
--ui-primary         --ui-primary-text
--ui-secondary       --ui-secondary-text
--ui-success         --ui-success-text
--ui-danger          --ui-danger-text
--ui-warning         --ui-warning-text

/* Neutrals */
--ui-bg              --ui-bg-subtle
--ui-surface         --ui-border
--ui-text            --ui-text-muted
--ui-text-inverse    --ui-overlay

/* Sizing & shape */
--ui-radius-sm       --ui-radius
--ui-radius-lg       --ui-radius-full

/* Shadows */
--ui-shadow-sm       --ui-shadow
--ui-shadow-lg       --ui-shadow-xl

/* Typography */
--ui-font            --ui-font-mono
--ui-font-size       --ui-line-height

/* Misc */
--ui-tooltip-bg      --ui-tooltip-text
--ui-select-arrow    --ui-transition
