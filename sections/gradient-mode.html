Gradient Mode

Overview

Gradient mode adds a subtle diagonal gradient to all colored components — buttons, toggles, checkboxes, radios, tags, pagination, progress bars, badges, sliders, and status dots. It works with every theme and adapts automatically to light/dark mode.

Activate via HTML attribute

Show code 
<!-- Enable gradient mode -->
<html data-ui-gradient>

<!-- Combine with a theme -->
<html data-ui-theme="ocean" data-ui-gradient>

Toggle via JavaScript

Show code 
// Toggle gradient on/off
document.documentElement.toggleAttribute('data-ui-gradient');

// Turn on
document.documentElement.setAttribute('data-ui-gradient', '');

// Turn off
document.documentElement.removeAttribute('data-ui-gradient');

Preview

Click the gradient toggle button (colored circle) in the top-right corner to toggle gradient mode and see the effect on all components.
